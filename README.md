## 📖 תיאור

הסרט שבראש הוא אפליקציה חינוכית המסייעת למורים ולתלמידים לפתח אסטרטגיות הדמיה מנטלית (Mental Imagery) להבנת הנקרא. האפליקציה משתמשת ב-AI ליצירת טקסטים, תרגילים, דפי עבודה ותרגילי זיהוי אווירה.

## ✨ תכונות

- 🎯 **מעבדת הקראה**: יצירת טקסטים עם אווירה ספציפית והקראה קולית
- 👩‍🏫 **מרחב המורה**: יצירת דפי עבודה מותאמים אישית עם שאלות הבנה
- 🎨 **תרגילי אווירה**: זיהוי וניתוח אווירה בטקסטים
- 🖼️ **יצירת איורים**: יצירת איורים אוטומטית למילים וביטויים
- 🔊 **קריינות**: הפקת קריינות קולית לטקסטים
- 📄 **ייצוא**: הדפסה וייצוא ל-Word ו-HTML

## 📤 העלאת הפרויקט ל-GitHub

### שימוש בסקריפט האוטומטי (מומלץ)

**ב-Windows (PowerShell):**
```powershell
.\deploy-to-github.ps1
```

**ב-Linux/Mac:**
```bash
chmod +x deploy-to-github.sh
./deploy-to-github.sh
```

הסקריפט ינחה אתכם בתהליך:
1. יצירת repository ב-GitHub
2. הגדרת remote
3. העלאת הקוד

### העלאה ידנית

1. צרו repository חדש ב-[GitHub](https://github.com/new)
2. הרצו:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

## 🚀 התקנה והרצה

### דרישות מוקדמות

- Node.js (גרסה 18 ומעלה)
- מפתח API של Google Gemini (קבלו מ-[AI Studio](https://aistudio.google.com/app/apikey))

### שלבי התקנה

1. **שכפול הפרויקט:**
   ```bash
   git clone <repository-url>
   cd הסרט-שבראש---אסטרטגיות-הדמיה
   ```

2. **התקנת תלויות:**
   ```bash
   npm install
   ```

3. **הגדרת מפתח API:**
   - העתיקו את `.env.example` ל-`.env.local`
   - הוסיפו את מפתח ה-API שלכם:
   ```
   API_KEY=your_actual_api_key_here
   ```

4. **הרצת האפליקציה:**
   ```bash
   npm run dev
   ```

5. פתחו את הדפדפן בכתובת: `http://localhost:5173`

## 🏗️ בנייה לפרודקשן

## 🎓 שימוש

### יצירת טקסט במעבדת הקראה

1. בחרו "נושא ל-AI" או "טקסט משלי"
2. (אופציונלי) בחרו אווירה לטקסט
3. לחצו "צור טקסט"
4. האזינו לטקסט והשתמשו באסטרטגיות הדמיה

### יצירת דף עבודה במרחב המורה

1. הדביקו טקסט או השתמשו בטקסט מהמעבדה
2. לחצו "צור דף עבודה"
3. ערכו, הוסיפו איורים, והפיקו קריינות
4. ייצאו ל-Word, HTML או הדפיסו

### תרגילי זיהוי אווירה

1. לאחר יצירת דף עבודה, לחצו "צור תרגילי אווירה"
2. פתחו את הטאב "תרגילי אווירה"
3. ענו על התרגילים וקבלו משוב מותאם
